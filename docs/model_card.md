# Model Card

## Model Purpose
Detect visual indicators of cheque fraud in mobile deposit images.

## Inputs
Cheque image, extracted regions, image metadata, customer history indicators, prior deposit image fingerprints.

## Outputs
Fraud probability score, reason codes, explainability heatmap, recommended action.

## Limitations
Performance depends on image quality, representative training data, and continuous monitoring. Human review should remain in the workflow for high-risk decisions.
